Issue 09 - Necron Warriors (1 of 2)

Magazine - 09

Magazine - 10

These two issues are each half of the boxed set which is in the small, medium and large starters as well. It looks like there are two sets of these in the contents listing photo - there are six scarab swarms anyway. You can build these Necron Warriors with either the “Gauss Flayer” (rifle) or the “Gauss Reaper” (shotgun) … unlike the previous models.

The (totally made up) rumour is that the Canoptek Scarabs (as well as Tyranid Ripper Swarms, Tau Drones and others) came as “pets” because the original kits were made during Pokemania.

Issue 13 - Space Marine Assault Intercessors

  • Magazine
  • Rules

  • This sprue is ONLY in the small Recruit starter, the medium Elite starter and the large Command starter sets.

  • The seargeant can be assembled as a mook - so you can use these to fatten up an existing squad

  • The seargeant can be assembled with a [Heavy Bolt Pistol]() or a [Plasma Pistol]()

    • There are specific head/helmet/sword bits for each
    • You need to take drastic steps to get an exposed head without a seargent’s helmet

If you want to go crazy the retail kit has two identical sprues to make a squad with. You could get one of those “full” kits, build them as two 5-man squads, then use this sprue to fatten up that squad.

When I was assembling these myself, I dropped the first helmet. I used the seargent’s mook-helmet as a replacement - the helmet “mounts” might be interchangeable.
