2020-10-19

I’ve found this information elsewhere and have been slow to update this page.

I’ll probably bring it up to date eventually, and, include my feelings on painting the figures.

The Mortal Realms is a partwork magazine run by Hatchette for Games Workshop. Here, I’m collecting an list of the contents, rules and links to the individual items. I also have a longer blogpost that I’ll update as I go.

ForbiddenPlanet sells the things, lets you pic-and-mix, ships internationally, ships sooner and can be cheaper than the producer … when you purchase at least 11 issues at once.1 I’m only putting titles and links on things that are in stock.

Delivered

Issue 01

Everyone panic bought this issue. There’s supposed to be another Chainrasp Horde on the way - you can see the not-blue ones in the promotional art.

  • 10 members of a Chainrasp Horde
    • the “leader” has an alternate build
    • two of these are in Soul Wars
    • one of these is in Tempest of Souls (and with paint)
    • this is available as an Easy to Build set
  • 3 Sequitors
    • the “leader” has an alternate head
    • this sprue is in Storm Strike
    • this is the sprue from Dreadfane
    • this is the sprue from Stormcast Eternals Paint Set
    • this is available as an Easy to Build set
  • 6 dice, a ruler and small battle mat
    • the dice are everywhere; they seem to be the new “semi translucent” material

Issue 02

  • 4 Myrmourn Banshees (from the Storm Strike and the Dreadfane sets)
    • this sprue is in Storm Strike
    • this is the sprue from Dreadfane
    • this is available as an Easy to Build set
  • Base / Corax White
    • this seems to be “normal” paint
    • it needed heavy shaking
  • Starter Brush
    • this is in the Age of Sigmar Paint & Tool Set
    • this is in the Stormcast Eternals Paint Set
    • this is in the Nighthaunt Paint Set

Issue 03

The first goodboi.

  • 4 Castigators
    • this sprue is in Storm Strike
    • this is available as an Easy to Build set
  • 1 Gryph-Hound
    • this sprue is in Storm Strike
    • this is available as an Easy to Build set
  • Metallic / Retributor Armour
    • this is in the Age of Sigmar Paint & Tool Set
    • this is in the Stormcast Eternals Paint Set

Issue 04 - Glaivewraith Stalkers

  • 4 Glaivewraith Stalkers
    • the “musician” has an alternate build
    • this sprue is in Storm Strike
    • this is the sprue from Nighthaunt Paint Set
    • this sprue is NOT in Soul Wars
      • the Soul Wars sculpts have slotted bases
  • Technical / Nihilakh Oxide
    • technically it’s the best colour
    • curiously, this isn’t in any of the starter paint sets

Issue 05 - Xandria Azurebolt

  • Xandria Azurebolt – Knight Incantor
    • this is the magazine’s exclusive mini
    • why not a ghost?

The legend is that this sculpt was done as a training exercise. It was given to a trainee sculptor as their first task and a throwaway piece. Nobody was expecting it to be worth producing … but it was quite good!

Issue 06

  • The Thorns of the Briar Queen warband
    • this is a “special” squad of 7 figures
    • these are the Nighthaunt warband in Nightvault
      • I’d be surprised if the Stormcast warband doesn’t show up as well
    • these are available on their own for Age of Sigmar

I found these a joy to paint/finish.

Issue 07

  • Metallic / Stormhost Silver
    • again, this isn’t in any of the starter paint sets
  • Shade / Reikland Fleshshade
    • it’s the non-Gloss variant
    • this is in the Stormcast Eternals Paint Set
    • this is in the Nighthaunt Paint Set

Issue 08

  • 1 Celestar Ballista team
    • this looks like a different sculpt from what’s in Soul Wars or Tempest of Souls
      • the heads, hands and sculpted bases are different
    • this is available as Easy to Build: Celestar Ballista on its own
  • Base / Kantor Blue
    • this is in the Age of Sigmar Paint & Tool Set
    • this is in the Stormcast Eternals Paint Set

Issue 09 - Sequitors and Castigators

  • 3 Sequitors
    • this is from a sprue in Soul Wars (where it uses grey plastic)
    • this is from a sprue in Tempest of Souls (where it uses gold plastic) (… and with paint)
  • 2 Castigators
    • this is from a sprue in Soul Wars (where it uses grey plastic)
    • this is from a sprue in Tempest of Souls (where it uses gold plastic) (… and with paint)

Issue 10 - Tomb Banshee

This is a delighful old vampire sculpt which comes with a square scenic base on the sprue and around one in the packet. Can be used to lead a Shriekr Host … if you get an extra pack of Myrmourn Banshees.

  • 1 Tomb Banshee
    • this is an old Vampire Counts sculpt with a square scenic base
    • this is sold on its own (but might not come with the round base)
  • Base / Khrone Red
    • this is in the Paint & Tool Set (… and with paint)

Issue 11

This is the first terrain issue and the first multi-part kit. The kit was previously called “Garden Of Morr” in the world that was.

  • 1⁄3 of the Sigmarite Mausoleum set (part 2 of 3)

Issue 12 - Knight-Questor

  • 1 Knight-Questor
    • this is the 2016 monopose hero
    • it’s also in the Hammerstrike Brethren set

Issue 13 - Discover the Forces of Order

This was the first issue released durring the COVID-19 Lockdown in the UK.

  • Brush / Citadel Medium Dry Brush
    • This is also in Complete Citadel Brush Set
    • This is also in Essential Citadel Brush Selection
  • Base / Mechanicus Standard Grey
  • Base / Abaddon Black
    • this is in the Age of Sigmar Paint & Tool Set
    • this is in the Stormcast Eternals Paint Set
    • this is in the Tempest of Souls & Paint Set

Issue 14

I wanted doubles of this to make a batallion … but … the magazine sold out before I ordered a second one.

  • 2 Dreadblade Harrows
    • this is sold on its own

Issue 15

  • 1⁄3 of the Sigmarite Mausoleum set (part 2 of 3)

Issue 16 - Sequitors and Castigators

  • 3 Sequitors
    • two of these are in Soul Wars
    • one of these is in Tempest of Souls
    • this is in the Tempest of Souls & Paint Set
  • 2 Castigators
    • two of these are in Soul Wars
    • one of these is in Tempest of Souls
    • this is in the Tempest of Souls & Paint Set
  • 6 Dice

Issue 17 - Dreadscythe Harridans

This was the first “BaseGate” issue - the included bases (were the correct size but) had an extra hole in them and people were grumpy.

  • 5 Dreadscythe Harridans
    • the retail set comes with 10 from two identical sprues

Issue 18 - Anvils of the Heldenhammer

This issue is cheaper to buy in parts - unless you really want a game mat.

  • Base / Dryad Bark
  • Metallic / Leadbelcher
    • this is in the Age of Sigmar Paint & Tool Set
    • this is in the Stormcast Eternals Paint Set
  • Game Mat

Issue 19

  • Knight of Shrouds
    • this is from a sprue in Soul Wars
  • Guardian of Souls (with Nightmare Lantern?)
    • this is from a sprue in Soul Wars
    • I believe that this variation is holding the “Nightmare Lantern” rather than the Mortality Glass
  • Spirit Torment
    • this is from a sprue in Soul Wars
    • this may or may not have been converted to carry grocery bags

Issue 20

This was the last issue listed on ForbiddenPlanet before the COVID-19 Lockdown caused Hatchette to slow down.

  • Lord-Arcanum (on Gryph-Charger)
    • this is from a sprue in Soul Wars
    • this is NOT [the one sold on its own][buyLordArcanum] - that one has a scenic base
  • Evocator-Prime
    • this is from a sprue in Soul Wars
    • I think that this model is just a unit leader
