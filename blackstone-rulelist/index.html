Blackstone Rule List

  • Reinforcements
  • Back Rules
  • Fore Rules

This is an attempt to “distill” the BlackStone Fortress rules into a more-dry form (like X-Wing or Dominion!) after several play sessions. Should make for quicker reference and play (if you’re like me) at the potential cost of being “scary” to people who like their information burried in prose.

I may-have “introduced” some terms … Chain, Precise, Injury etc.

Reinforcements

  • Hostiles which were not deployed (due to running out of minis) cound as “slain” along with any “actually” slain models.

  • If any hostiles from the group were slain; you make a reinforcement roll

    • we also do it if any weren’t deployed
    • we (used to just) do it at the end of a turn
    • we (should now) do it at the end of the hostile’s group’s activation
    • you don’t exceed the group’s sizes

  • use the smallest slain hostile in the group, and, roll on this chart to see how many slain (missing?) hostiles are returned

  • returned hostiles are placed next to a portal (one at a time)
    • if there are group survivors on the board; the portal closest to the survivors is used
    • if there are no group survivors; the portal closest to an explorer is used
    • exploers may now overwatch
  • after each hostile is returned it attacks or moves

    • you never exceed the group’s sizes

Back Rules

Advance
The hostile model moves towards the nearest explorer model.
If the hostile can attack; they attack the nearest explorer model.
If the hostile can’t attack any explorer; their activation ends.
Aim
Action which costs 1+ to make the explorer’s next attack is “Aimed.”
The attack ignores cover.
Annihilate
Hostile needs to be adjacent to an explorer.
If the hostile is not Empowered then they become Empowered.
The hostile then performs a Precise attack on an adjacent explorer.
Bellowing Beast
Actions cost 1 more for adjacent explorers, up to 6+.
This effect doesn’t stack regardless of how many Beastmen are present.
Charge
the hostile moves towards the nearest explorer
if the hostile can attack; they attack the neatest explorer
if the hostile can’t attack; they again move towards the nearest explorer
Chain n
Make n attacks
Successive attacks must be against targets within 1
Cover
Wounds are “downgraded” against a target in cover.
Discard
Resource cards that’re “discarded” are returned to the ship’s deck once used
Disrupt
If there are unspent activation or destiny die; hostile player discards one
If there are no unspent activation or destiny die; hostile player wounds an explorer
Empowered
“twin roll” attach dice
Is lost when suffering Wound (and not saving it)
Empower
If they’re not Empowered, they become Empowered
If they are Empowered, treat this as a Disrupt
Exhaust
Resource cards that’re “exhausted” can’t be re-used until the end of the turn/activation
Fall Back
move the hostile piece up-to double its movement
they must try to end their move out of sight of any/all/nearest explorer
they should try to move away from any/all/nearest explorer
Furious Charge
+1 to Move value when Charging
Hold
Do nothing.
Hide in the Shadows
Counts as in cover if attacked from > 3 hexes away
Injury
what I call a “not grievous wound”
Mortal
what I call a “grievous wound”
Counts as three Injuries, but, for accounting reasons should use a red token.
Move Up
the weapon cannot be fired at this range
the model moves towards the closest enemy instead
Onslaught
Attack the closest Vulnerable explorer.
Then, attack the closest Vulnerable explorer - even if that’s now a different explorer.
Piercing
Attack ignores saves
Precise
Re-rolls missed attacks
Recover (4+ then Vitality) (else Action)
If the model has an Injury, remove it
If the model doesn’t have an Injury - there will be another action
Shield
It’s a 20+ FNP roll for hostiles.
It negates the whole wound Injury or Mortal.
Stun
if a model is “stunned” turn it on its side
if a model is already stunned, the second instance of the effect is another wound
if it’s an explorer - it must spend an action die to “stand up” before doing anything
if it’s a hostile - it must spend its whole activation to stand up
Threat Level
set to 0 at start of each combat
if a Spindle Drone is slain; threat level increases
can’t exceed 3
Vulnerable
Consider an enemy as “Vulnerable” if they’re within range and visible.
Sneak
the hostile musn’t be visible to any explorer
Make a move that ends as close as possible to an explorer without entering a hex visible to an explorer.
Splash
make an attack roll against all enemies in the target’s hex

Fore Rules

These combine, modify, or parameterize Back Rules

Alert
If Threat Level < 3 then increase threat level by 1 and Onslaught
If Threat Level = 3 then Precise Onslaught
Blast
Piercing Splash
Frenzied
Chain 3
Fury
Precise Onslaught (Negvolt; Aim Piercing Precise Onslaught)
Lightning Arc
Aimed Piercing
Lord of War
Act/Behave twice
Overcharge
Piercing attack against closest enemy
Pounce
If there is an adjacent enemy; Precise Onslaught
If there is no adjacent enemy; Precise Charge
Rapid Fire
Precise Onslaught
Recharge (else Action)
Recover else Advance
Regenerate
Recover else Disrupt
Rush
Move towards the closest explorer, then Charge
Thunder Hammer
Piercing
Traitor Guardsmen Sergeant
Model has +1 Wound
Traitor Guardsmen Groups
Max 1 Sergeant
Max 1 Flamer
If group has >= 5 models; one must be a Sergeant and one must have Flamer
Voltagheist Field
Shield
gets +2 for each other Negavolt Cultist within 1 tile
Psychic Shield
Shield
It gets a +4 if the model is Empowered
